pyjns.k30 module¶

K30 모듈¶

설명¶

SenseAir 사의 이산화탄소 농도 센서인 K30 STA를 조작할 수 있는 모듈이다.

참고¶

관련링크 * http://www.senseair.com/products/oem-modules/k30/

작성자¶

  • JoonYong (joonyong.jinong@gmail.com)
class pyjns.k30.K30(config)¶

Bases: object

K30 적외선 온도센서의 RS485 프로토콜을 구현한 클래스이다. _로 시작하는 메소드는 private 로 간주하고, all 이 붙어있는 메소드는 연결된 전체 센서에 대한 명령으로 이해하면 된다.

>>> config = {"tty" : "/dev/ttyUSB1", "retry" : 3}
sensor = K30 (config)
print sensor.readCO2 ()
